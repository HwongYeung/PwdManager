# 安全说明

## 加密实现

本项目使用简化的加密方案，适用于微信小程序环境：

### 加密算法
- **XOR加密**：基于密钥的对称加密算法
- **Base64编码**：对加密结果进行编码存储
- **密钥派生**：从PIN码和盐值派生加密密钥

### 安全特性
1. **PIN码保护**：6位数字PIN码作为主密钥
2. **随机盐值**：每次设置PIN码时生成随机盐值
3. **密钥派生**：通过多次哈希增强密钥强度
4. **本地存储**：数据仅存储在用户设备本地

### 安全限制

⚠️ **重要提醒**：
- 本项目使用的是简化加密方案，主要用于演示和学习
- XOR加密相对简单，不如AES等标准加密算法安全
- 建议在生产环境中使用更强的加密算法

### 安全建议

1. **定期更换PIN码**：建议定期更换PIN码
2. **备份重要数据**：定期导出重要密码数据
3. **安全环境使用**：避免在不安全的网络环境中使用
4. **妥善保管PIN码**：忘记PIN码将导致数据无法恢复

### 未来改进

如需更高安全性，可考虑：
- 集成WebCrypto API（如果小程序支持）
- 使用更强的加密算法
- 实现更复杂的密钥派生函数
- 添加数据完整性校验

## 隐私保护

- 不收集任何用户数据
- 不连接外部服务器
- 所有数据仅存储在本地
- 不使用任何追踪技术
